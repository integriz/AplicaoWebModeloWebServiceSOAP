# AplicaoWebModeloWebServiceSOAP 

Trata-se de uma pequena aplicação modelo com alguns endpoints Soap com autenticação e envio e recebimento de arquivo.

Gerado no NetBeans 8.2 com o servidor Glassfish 4.1.1

Java 8 (JAX-WS) 100.0%

Obs.: O projeto ClienteWebServiceSoap disponível no Git é o cliente modelo deste projeto.
